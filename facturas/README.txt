Módulo de facturas para Odoo

Este módulo de facturas para Odoo permite a los usuarios crear y gestionar facturas de clientes y proveedores en su instancia de Odoo. 
El módulo incluye vistas de formulario, kanban y calendario para visualizar las facturas, y también cuenta con funcionalidades para 
clasificar las facturas como "a pagar", "a cobrar" o "borrador". 

Autor: Miguel Fernández Fernández

Instalación:

1. Descarga el archivo ZIP del módulo de facturas.
2. Inicia sesión en tu instancia de Odoo como administrador.
3. Haz clic en el icono de configuración en la esquina superior derecha y selecciona "Aplicaciones".
4. Haz clic en el botón "Cargar módulo".
5. Se abrirá una ventana emergente. Haz clic en el botón "Seleccionar archivo" y selecciona el archivo ZIP de tu módulo.
6. Haz clic en el botón "Cargar".
7. Si el archivo ZIP se ha cargado correctamente, deberías ver un mensaje que indica que el módulo de facturas ha sido instalado con éxito.
